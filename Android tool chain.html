ADP的安装

启动eclipse，点击window——>preferences——>install/update——>available software sites，点击Add，把如下几个网址添加过去

  • Eclipse GEF http://download.eclipse.org/tools/gef/updates/releases/
  • Eclipse EMF http://download.eclipse.org/modeling/emf/updates/releases/
  • Eclipse GMF http://download.eclipse.org/modeling/gmf/updates/releases
  • Eclipse Webtools http://download.eclipse.org/webtools/updates/
  • Google eclipse Plugin http://dl.google.com/eclipse/plugin/3.5

[http://stackoverflow.com/questions/4249695/adt-requires-org-eclipse-wst-sse-core-0-0-0-but-it-could-not-be-found ADT requires 'org.eclipse.wst.sse.core 0.0.0' but it could not be found]

  1. help->install new software
  2. click on "available software sites"
  3. select all entries (CTRL-A) and remove them
  4. add new entry at location http://download.eclipse.org/releases/indigo/
  5. now add again the ADT location http://dl-ssl.google.com/android/eclipse/

Android Source的获取

  • Step 1. 编辑 /home/username/bin/repo文件

REPO_URL='git://android.git.kernel.org/tools/repo.git' 修改为REPO_URL='http://android.git.kernel.org/tools/repo.git'

  • Step 2. 运行新的命令

$ repo init -u http://android.git.kernel.org/platform/manifest.git -b gingerbread

  • Step 3. 修改文件.repo/manifests/default.xml

将fetch="git://android.git.kernel.org/" 改成fetch="http://android.git.kernel.org/"

  • Step 4. 运行命令

repo sync

JDK的安装

如果你的电脑上已经安装了JDK，请确认它的版本符合需求。JDK 1.4或者GCJ(Gnu Compiler for Java) 均不支持Android开发。推荐安装最新的JDK 6。如果你需要在Fedora上安装Sun JDK 6，可以参考这篇日志。 

#切换到root用户
su

#安装JDK
sh /some/path/jdk-6u22-linux-i586-rpm.bin

#安装java, javac等
## java ##
alternatives --install /usr/bin/java java /usr/java/jdk1.6.0_27/jre/bin/java 20000
## javaws (32-bit only) ##
alternatives --install /usr/bin/javaws javaws /usr/java/jdk1.6.0_27/jre/bin/javaws 20000
## Firefox浏览器插件32位
alternatives --install /usr/lib/mozilla/plugins/libjavaplugin.so libjavaplugin.so /usr/java/jdk1.6.0_27/jre/lib/i386/libnpjp2.so 20000
## Firefox浏览器插件64位
alternatives --install /usr/lib64/mozilla/plugins/libjavaplugin.so libjavaplugin.so.x86_64 /usr/java/jdk1.6.0_27/jre/lib/amd64/libnpjp2.so 20000
## 安装javac
alternatives --install /usr/bin/javac javac /usr/java/jdk1.6.0_27/bin/javac 20000

#切换默认Java # 运行下面的命令，选择刚刚安装的Sun JDK
alternatives --config java

#查看现在的Java版本，看看是否切换成功
java -version

这里有一个很重要的命令alternatives，如果你对该命令还比较陌生，点击linux command。
