Launcher Menu 的安装

  • 1. git clone git://github.com/AnderWeb/android_packages_apps_Launcher.git
  • 2. Create a Android project with eclipse(Using existing project)
  • 3. Add Build Path for Launcher project.
    • 1)framework_intermediates/classes.jar :这个主要是android的框架类
    • 2)android-common_intermediates/classes.jar :这个包含com.android.common.Search这个类
    • 3)core_intermediates/classes.jar :这个包包含dalvik.system.VMRuntime这个类

the upon jar packages could be getting from compiled android source code

  • 4. Rename package
  • 5. Then check the code without error, if the error is happaned at @Overwite, comment it
  • 6. Run launcher with android simulator.
